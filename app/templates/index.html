Key Concepts

Discover the concepts of machine learning, text generation, and web development!

NLP

Natural Language Processing (NLP) is a subset of computer science that involves teaching AI models to process, understand, and communicate in human language.

GPT

Generated Pre-trained Transformer (GPT) is a language model that processes and generates text. The latest versions are GPT-2 and GPT-3.

AITEXTGEN

AiTextgen is a Python-based tool used for training AI models and generating text using GPT-2. With preset parameters of 124M, 355M, and 744M, aitextgen is able to train and generate text at a relatively quick pace.

FLASK

Flask is a Python-based web framework that simplifies the process of backend web development. Flask is a microframework that is simple to use because it provides the tools, libraries, and technologies to build a web application.

HTML

HyperText Markup Language (HTML) is a programming language used to create the frontend (what a user is able to see) of a document designed for display in a web browser.

CSS

Cascading Style Sheets (CSS) is used to create the display of a website. It can be assisted by HTML and JavaScript to make the website more interactive.

CLEANING THE DATA

The Trump AI was trained using a dataset consisting of Trump Twitter Archive data. The dataset contained several emojis, hashtags, mentions and other unwanted characters. Using regular expressions, we were able to clean the dataset and use it to train the model more effectively.

BEFORE

Before, the dataset contained several hyperlinks, hashtags, and mentions.

AFTER

After cleaning, the dataset looks far more organized! With the cleaned dataset, we trained the model to speak more coherently!
